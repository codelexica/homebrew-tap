# Codelexica Tap

## How do I install these formulae?

`brew install codelexica/tap/<formula>`

Or `brew tap codelexica/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "codelexica/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
